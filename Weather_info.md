# 각 구단 야구장 날씨 정보 서비스

구단명을 검색 조건으로 하여 대응되는 야구장의 **강수량, 기온(최저, 최고), 풍속, 습도 등** 정보를 **현재 시간 기준 +7일까지** 조회할 수 있는 야구장 날씨 정보 서비스

## API 기본정보

-   **요청주소 (Request URL)**\
    `http://apis.data.go.kr/1360000/MidFcstInfoService/getMidFcst`

-   **서비스URL (Service URL)**\
    `http://apis.data.go.kr/1360000/MidFcstInfoService`


## 📥 요청변수 (Request Parameter)


| 한글 설명 | 필드명 | 비고 |
|-----------|--------|--------|
| 서비스키 | ServiceKey | 공공데이터포털에서 받은 인증키 |
| 페이지 번호 | pageNo | 페이지 번호 |
| 한 페이지 결과 수 | numOfRows | 한 페이지 결과 수 |
| 응답자료형식 | dataType | 요청자료형식(XML/JSON), Default: XML |
| 지점번호 | stnId | 108 전국, 109 서울·인천·경기 등 |
| 발표시각 | tmFc | 일 2회(06:00, 18:00), 예: YYYYMMDD0600 / YYYYMMDD1800 — 최근 24시간 자료만 제공 |
